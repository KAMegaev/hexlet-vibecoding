# hexlet-vibecoding

Учусь вайбкодингу, тут будут мои эксперименты.

## Калькулятор калорий

Простое веб-приложение для расчёта суточной нормы калорий и БЖУ.

### Возможности

- Расчёт базового обмена веществ (BMR) по формуле **Миффлина — Сан Жеора**
- Учёт уровня физической активности (TDEE)
- Корректировка под цель: похудение, поддержание или набор массы
- Распределение белков, жиров и углеводов (30% / 30% / 40%)

### Запуск

Это статическое приложение без сборки и зависимостей. Откройте `index.html`
в браузере либо запустите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

### Файлы

- `index.html` — разметка и форма ввода
- `styles.css` — оформление
- `script.js` — логика расчётов
